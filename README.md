# TL;Pinch

*Pinch to see less, know more.*

TL;Pinch is an AI-powered Safari extension and iOS app that lets you summarize, explain, and ask questions about any web page with a simple pinch gesture.

## Features
- **Pinch gesture detection:** Trigger actions with a pinch gesture on any web page (works reliably across tabs).
- **AI-powered summarization:** Instantly generate a TL;DR summary of the current page using an AI backend.
- **Explain selected text:** Select any text and get an AI-generated explanation.
- **Questions & answers:** Automatically generate relevant questions and answers about the page content in one API call.
- **Modern popup UI:** Clean, accessible popup with animated loading spinner and smooth scrolling (macOS-friendly).
- **Localized extension name/description:** Uses _locales for proper browser localization.
- **Pinch icon branding:** Extension icons updated to use the Pinch brand.
- **Robust manifest:** MV3-compliant, secure permissions, and future-proofed for new features.
- **No legacy or unused files:** Removed rules, new tab page, and metadata for a clean codebase.
- **Programmatic content script injection:** Ensures pinch detection works on the active tab, even with multiple tabs open.
- **Production backend integration:** All AI features use the deployed backend API.
- **iOS app onboarding:** Native onboarding and launch experience for iOS users.
